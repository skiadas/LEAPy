The StudentList Class¶

class StudentList(lst)[source]¶

Bases: list

Initialize a StudentList object using provided data. The provided object needs to act as a list of dictionaries. Those dictionaries are passed to the student.Student class to form the individual student objects. A typical way to obtain such a list is via a dataset_utils.Dataset object.

Use update_from_dataset() to add new key entries to the students, meth:perform_placement to place them according to the rules set (see my_utils.Placement), meth:dataset to convert to a spreadsheet/dataset format, export_to_template() for generating reports.

update_from_dataset(key, sheet, force_update=False, target_key=None)[source]¶

Updates any students common to self and sheet with the information in them. Will use the values described by key and target_key respectively to identify which students in the two datasets match. sheet needs to be dataset_utils.Dataset object. If force_changes is True, the method will update the values of any keys that are present in sheet, overwriting any possibly existing values. This will rarely be what you want. If target_key is None, it will use the same key on both datasets. The conversions in my_utils.standardize_key() are in place.

perform_placement()[source]¶

Simple wrapper for student.Student.perform_placement(). Attempts to assign a placement to every student.

dataset(keys=[])[source]¶

Returns self in a dataset format. Uses all keys present unless a list is specified.

export_template(template, format=None, sortby=None)[source]¶

Exports the student information according to the template specified. LaTeX or HTML output is possible by setting format='tex' or 'html'. If no format specified, and the template is a filename string, it will use the extension to determine the format. Returns the output stream. See Creating Templates for more information on templates.
