The Dataset Class¶

class Dataset(data)[source]¶

The Dataset class models a basic dataset/spreadsheet functionality. It can read and write Excel sheets, CSV and HTML tables. It implements a basic iteration over its rows, where each row is returned as a dictionary, with the variable/column names as keys.

get_col(col)[source]¶

Returns the columns described by col. Accepts column descriptions as either a number starting at 0 or column name, or as a list of the same.

get_row(row)[source]¶

Returns the rows described by row. Accepts row descriptions as either a number or a list of numbers.

classmethod Read(dataset, filetype=None, raw=False)[source]¶

Reads datasets in either an Excel or a CSV form, or from a HTML table. Expects first row to be variable names, every subsequent row to be data, and that no extra rows are present. Returns the result as a Dataset object. Only understands character and numeric data at this point.

classmethod Read_survey()[source]¶

Helper method, that reads the data from the math survey that students have to take.

export(file, keys=[], key_names=[], format=None, *args, **kwds)¶

Exports the dataset to an appropriate format, automatically detected by file if it is an actual filename. If no keys are provided, it will use all of them, this is not typically what you want. You may provide key_names to have those appear at the column headers instead.

subset(row_tester)¶

row_tester is a function that takes a dictionary representing a row, and returns whether that row should be included in the subset.

fix_col(lst)[source]¶

A helper function attempting to uniformize the items in a list (data column). If a column consists of only strings representing numbers, turns them to numbers. Treats empty strings as missing values and replaces them with None.
